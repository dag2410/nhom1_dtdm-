🚀 Triển khai deploy bằng Vercel

Website này được triển khai trên Vercel theo các bước sau:

  • Khởi động Cloud Shell và tạo project.
  • Tạo App Engine với vùng asia-southeast1.
  • Tạo file HTML và file cấu hình app.yaml.
  • Chạy lệnh gcloud app deploy để triển khai.
  • Truy cập website bằng lệnh gcloud app browse.

Đây là ví dụ điển hình về mô hình PaaS, giúp triển khai ứng dụng mà không cần quản lý hạ tầng.
